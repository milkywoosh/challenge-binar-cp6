**HOW TO START THIS SIMPLE APP**
> Just follow those instructions:
```
1. Open your lovely terminal
  1.1. npm i -> to download nodes package
2. isi .env file -> setting your db, user, and password on postgres
3. make sure start your postgres database on terminal
4. sequelize db:create -> create new db in postgres
5. sequelize db:migrate -> create model directly in database
6. port 3050 -> localhost for URL
7. npm run start -> start the apps to run on the localhost

"locahost:3050/api/usergames/join-table" --> first api to start
```
